# Python仓库管理系统 python878

本项目是基于Python开发的仓库管理系统，集成现代化的前后端技术栈，致力于提高仓库管理的效率和准确性。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue

## 功能模块

系统主要包括以下功能模块：

- 商品管理
  - 管理商品的详细信息
- 分类管理
  - 对商品分类进行维护
- 用户管理
  - 管理系统用户的账号和权限
- 日志管理
  - 记录和查询系统操作日志
- 系统信息
  - 查看系统的基础信息
- 批量删除
  - 支持批量删除操作，提高操作效率
- 分页查询
  - 支持分页显示数据，提升大数据处理的性能

## 系统角色

- 系统管理员：负责系统的维护和用户管理
- 普通用户：进行日常的仓库操作和管理

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js 2.x

## 目录结构

```
project/
│
├── backend/              # 后端代码目录
│   ├── apps/             # Django应用目录
│   ├── manage.py         # Django项目管理脚本
│   └── ...
│
└── frontend/             # 前端代码目录
    ├── src/              # Vue源码目录
    └── ...
```

## 部署步骤

1. 准备运行环境：安装Python、Django、MySQL和Vue.js等必要的软件
2. 部署后端服务：配置数据库，运行`manage.py`进行迁移和启动服务
3. 部署前端服务：构建Vue项目，放置到合适的静态文件服务器目录
